# File Organiser Project

This Python project automatically organises files in a folder based on their file type.

## Features
- Sorts files into categories (Images, Documents, etc.)
- Automatically creates folders if they don’t exist
- Handles unknown file types by placing them in an "Other" folder

## Technologies Used
- Python
- os module
- shutil module

## How to Use
1. Set the folder path in the script
2. Run the program
3. Files will be automatically organised

## Example
Before:
- test1.txt
- image1.jpg

After:
- Documents/test1.txt
- Images/image1.jpg
